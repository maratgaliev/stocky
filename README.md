# stocky

> Get your favourite stocks prices into your command line

## Install

```bash

npm i -g stocky

```

## Usage

  
```bash
# colors are: 'red', 'green', 'yellow', 'blue', 'magenta', 'cyan', 'white', 'gray'
stocky -s VTI VUN.TO VXUS -c "cyan" -i

```
